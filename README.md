# aitoon-vertical-director

> Claude Cowork용 세로 스크롤 웹툰 자동화 스킬 (Track B)

스토리를 넣으면 세로 웹툰용 컷 프롬프트를 자동으로 만들어주고, 이미지가 나오면 칸새 삽입·합본·네이버 업로드본 변환까지 처리합니다.

---

## ✨ v1.6에서 새로워진 점

- **시작에서 원고 종류를 고릅니다.** 스킬을 켜면 선택 카드 위젯이 떠서 **클린 원고**(말풍선 없이 컷만)와 **대사 포함**(말풍선까지) 중 하나를 위젯 안에서 고르고 진행합니다.
- **대사 포함(말풍선) 모드 추가.** 감정에 맞는 말풍선 종류를 프롬프트에 박아 **GPT가 말풍선까지 직접 그려줍니다.** 세로 특성에 맞춘 **빈 띠 규칙**(말풍선이 옆 컷 그림을 침범하지 않고 칸새 자리를 비워 둠)으로, 나중에 칸새를 넣어도 깔끔히 분리됩니다. 기존 **클린 원고 경로는 그대로 보존**됩니다.
- **네이버 업로드본 변환 추가.** `naver_export.py`로 최종 원고를 **가로 690px · JPG**(네이버 규격)로 일괄 변환합니다.
- **일관성 가드 강화.** 캐릭터 **머리색·의상색 고정**, **패널 테두리** 지정(무테 방지), 다중 인물 **첨부 서수 1:1 매핑**을 명세에 박았습니다.
- **칸새 스크립트 인자 형식 명확화.** `--cut-gaps`의 구분 규칙(장 안=콤마 `,` / 장 사이=파이프 `|`)을 `--help`·가이드·README에 통일했습니다.

---

## 전체 흐름

```
시작 카드에서 원고 종류 선택 (클린 원고 / 대사 포함)
    ↓
스토리 입력
    ↓
[스킬] 씬·컷 구성 + GPT 복붙 프롬프트 생성 + 칸새 간격 가이드
    ↓
[유저] GPT(나노바나나)에 복붙 → 컷 이미지 생성 → 작업 폴더에 저장
    ↓
[스킬] 칸새 자동 삽입 → 장별 편집본 생성 (gutter_inserter.py)
    ↓
[유저] (클린 원고면) 말풍선·효과음 수동 편집 / (대사 포함이면) SFX만
    ↓
[스킬] 장별 이미지 세로 합본 (merge_jang.py)
    ↓
[스킬] 네이버 업로드본 변환 — 가로 690px·JPG (naver_export.py)
```

---

## 특징

- 한 장(9:16)에 컷 2~3개를 세로로 묶어 GPT에서 뽑는 프롬프트 자동 생성
- 미드저니 sref 스타일 경로 / GPT 단독 경로 선택 지원
- 원고 종류 선택 — 클린 원고(말풍선 없음) 또는 대사 포함(GPT가 말풍선까지 그림, 세로 빈 띠 규칙)
- seam carving 기반 칸새 자동 삽입 (`gutter_inserter.py`)
- 편집 완료 후 세로 합본 자동화 (`merge_jang.py`)
- 네이버 업로드본 변환 — 가로 690px·JPG, 5MB 가드 (`naver_export.py`)

## 설치

Claude Cowork 데스크탑 앱에서 `.skill` 파일을 설치합니다.

1. [Releases](../../releases)에서 최신 `.skill` 파일을 다운로드
2. Cowork 앱 → Settings → Capabilities → 스킬 설치
3. 설치 후 대화에서 "세로 웹툰 만들어줘"로 시작

## 사용 방법

### 스킬 시작

```
"이 스토리로 세로 웹툰 만들어줘"
"세로 스크롤 웹툰 프롬프트 뽑아줘"
"aitoon-vertical 시작"
```

스킬이 시작되면 시작 카드로 **원고 종류(클린 원고 / 대사 포함)**를 먼저 고르고, 이어서 미드저니 sref 스타일이 있는지 물어봅니다.

### 경로 선택

| 경로 | 설명 |
|------|------|
| 미드저니 경로 | sref 그리드 이미지 분석 → MJ로 캐릭터 시트 생성 → GPT 컷 생성 |
| GPT 경로 | GPT로 캐릭터 시트 생성 후 컷 생성까지 GPT 단독 진행 |

### 산출물

- 캐릭터 ref 프롬프트 (GPT 복붙용)
- 장별 컷 생성 프롬프트 (GPT 복붙용, 클린 원고 또는 💬 말풍선 포함)
- 칸새 간격 가이드 (`1컷-(700px)-2컷-(1200px)-3컷` 형식)
- 네이버 업로드본 (가로 690px·JPG)
- 이미지 업로드 시 이상한 컷 진단 + 프롬프트 수정 (Step 7)

## 스크립트

### gutter_inserter.py — 칸새 자동 삽입

컷 이미지에 칸새를 자동 삽입해 장별 편집본을 만듭니다.  
seam carving으로 컷 경계를 찾아 칸새를 정밀하게 삽입합니다.

```bash
python scripts/gutter_inserter.py \
  --input ./ep08/ \
  --cuts "3,3,2" \
  --cut-gaps "700,1200|700,900|600" \
  --jang-gaps "300,300" \
  --margin-top 700 \
  --margin-bottom 700 \
  --out ./output/
```

> ★ 인자 형식: 한 장 안 컷 사이는 **콤마(`,`)**, 장과 장 사이는 **파이프(`|`)**. 거꾸로 넣지 않기. 자세한 변환 규칙은 `reference/gutter-guide.md`의 "스크립트 인자로 변환" 참고.

```bash
python scripts/gutter_inserter.py --help
```

**주요 옵션**

| 옵션 | 설명 |
|------|------|
| `--input` | 원본 컷 이미지 폴더 또는 콤마구분 파일들 |
| `--cuts` | 각 장의 컷 개수, 콤마 구분 (예: `"3,2,3"`) |
| `--cut-gaps` | 장별 컷 사이 칸새. 장 구분=`\|`, 장 안=`,` (예: `"700,1200\|600"`) |
| `--jang-gaps` | 장과 장 사이 칸새, 콤마 구분 (장 수 - 1개, 예: `"300,300"`) |
| `--margin-top` | 첫 장 상단 여백 픽셀 |
| `--margin-bottom` | 마지막 장 하단 여백 픽셀 |
| `--out` | 출력 폴더 경로 |

칸새 값은 스킬의 Step 5 칸새 간격 가이드에서 그대로 옮겨 사용합니다.

---

### merge_jang.py — 세로 합본

말풍선·효과음 편집이 끝난 장별 이미지를 세로로 이어붙여 합본 PNG를 만듭니다.  
칸새는 gutter_inserter가 이미 절반씩 넣어두었으므로 단순 concat으로 맞물립니다.

```bash
# 폴더 자동 수집 (파일명에 'edit' 포함, 자연 정렬)
python scripts/merge_jang.py --input ./ep08/ --out ep08_합본.png

# 파일 직접 지정 (순서 = 입력 순서)
python scripts/merge_jang.py ep08-1-edit.png ep08-2-edit.png ep08-3-edit.png --out ep08_합본.png
```

```bash
python scripts/merge_jang.py --help
```

---

### naver_export.py — 네이버 업로드본 변환

칸새 편집이 끝난 최종 원고(장별 편집본 또는 합본)를 **네이버 업로드 규격(가로 690px · JPG · 이미지당 5MB)**으로 일괄 변환합니다. 가로 690px LANCZOS 리샘플 + 화질우선 JPEG(q95)로 화질을 최대한 유지하고, 5MB를 넘기면 품질을 자동으로 낮춰 맞춥니다. 그래도 길면 `--max-height`로 세로 분할합니다.

```bash
# 장별 편집본 폴더를 통째로 변환 (출력: 입력폴더/naver/)
python scripts/naver_export.py --input ./ep08/

# 합본 1장을 변환하되 너무 길면 8000px씩 분할
python scripts/naver_export.py --input ep08_합본.png --max-height 8000
```

```bash
python scripts/naver_export.py --help
```

| 옵션 | 설명 |
|------|------|
| `--input` | 변환할 이미지 폴더 또는 콤마구분 파일들(`*-edit-*.png`, 합본 등) |
| `--out` | 출력 폴더(생략 시 입력 폴더 아래 `naver/`) |
| `--width` | 가로 폭(px). 기본 690(네이버 규격) |
| `--quality` | 시작 JPEG 품질. 기본 95(화질 우선) |
| `--max-mb` | 이미지당 최대 MB. 기본 5.0(네이버) — 초과 시 품질 자동 하향 |
| `--max-height` | (선택) 이 높이(px)로 세로 분할 |

## 범위

**이 스킬이 하는 것:**  
씬·컷 구성 / 세로 연출 판단 / GPT 복붙 프롬프트 생성(클린 원고 또는 선택적 말풍선) / 칸새 간격 가이드 / 칸새 자동 삽입 / 세로 합본 / 네이버 업로드본 변환(690px·JPG) / 이상한 컷 진단 및 프롬프트 수정

**이 스킬이 하지 않는 것:**  
이미지 직접 생성 / 말풍선·SFX 픽셀 합성(단 대사 포함 분기는 GPT가 말풍선을 직접 그림) / SVG 레이아웃·뷰어 / 페이지형 가로 그리드 컷만화
